# Проект 4: Место

## Mesto

Импортозамещение Instagram, пользователи будут делиться фотографиями, нажимать символ "Любо" под ними, выражая свою симпатию. 

### Используемые технологии
* Адаптивные свойства (max-width, min-width);
* HTML, CSS;
* JavaScript;
* Webpack, Babel.
